## Attribution template

For each mirrored skill, store on the `skills` row:

- `upstream_url` — the repo URL
- `original_author` — the author name above
- `license_spdx` — the SPDX ID (already in schema)
- Long description should prepend: *"Mirrored from [owner/repo](url) by {author}, licensed under {SPDX}. Originally surfaced via skills.sh."*

The `LICENSE` file from the upstream repo MUST be shipped inside the mirrored bundle (R2 object) so the license text travels with the skill per MIT/Apache-2.0/BSD terms.
